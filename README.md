# Jogos#
5 temas × 3 níveis = 15 fases jogáveis
Grid 8×8 / 10×10 / 12×12 com palavras em 8 direções
Seleção por arrastar com snapping de ângulo (toque e mouse)
Timer com urgência visual nos últimos 15s
Sistema de combo — acertos consecutivos multiplicam moedas
Moedas persistidas no localStorage
Estrelas baseadas em erros
Vitória / Derrota / Pausa / Retry / Próximo nível
Células com glow neon por cor de palavra encontrada
