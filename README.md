# Elo Palavras 🧩

detiillimichel-max.github.io/Jogos-Sonnet-Jogos/

O **Elo Palavras** é um jogo de caça-palavras dinâmico, desenvolvido com foco em performance, responsividade e uma interface moderna utilizando *Glassmorphism*.

## 🚀 Funcionalidades

* **Temas Diversos:** 5 categorias (Animais, Frutas, Esportes, Países, Cores).
* **Progressão:** 3 níveis de dificuldade por tema, com grids modulares (8×8, 10×10, 12×12).
* **Experiência de Jogo:**
    * Seleção intuitiva por arraste (mouse e touch) com *snapping* inteligente de direção.
    * Sistema de **Combo** para multiplicar moedas.
    * Timer com alerta de urgência visual nos últimos 15 segundos.
    * Feedback visual com *glow neon* por palavra encontrada.
* **Persistência:** Progresso e moedas salvos localmente (`localStorage`).
* **Gamificação:** Sistema de estrelas baseado no desempenho e erro zero.
* **PWA Ready:** Suporte para instalação e funcionamento offline via Service Workers.

## 🛠 Tecnologias Utilizadas

* **Linguagens:** HTML5, CSS3, JavaScript (ES6+).
* **Arquitetura:** Engine modular baseada em estados para garantir estabilidade.
* **Design:** CSS moderno com foco em experiência do usuário e interface fluida.

## 📱 Como Jogar

1. Acesse o projeto pelo seu navegador (Desktop Mobile ou pelo aplicativo de mensagens https://oio-fam.lovable.app/) detiillimichel-max.github.io/Jogos-Sonnet-Jogos/
2. Escolha uma categoria na tela inicial.
3. Arraste o dedo ou o mouse sobre as letras para formar as palavras listadas.
4. Tente completar o grid antes que o tempo acabe e sem cometer muitos erros para ganhar 3 estrelas!
